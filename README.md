基于 Windows API 与 AI 协同开发的桌面图标贪吃蛇
🛠️ 运行环境与准备工作在启动这个“桌面大胃王”之前，请确保你的电脑满足以下条件：操作系统：Windows 10 / Windows 11。Python 版本：Python 3.7 及以上。
依赖库安装：游戏高度依赖 Windows 底层接口，运行前请在终端（CMD 或 PowerShell）中执行以下命令安装核心支持库：Bashpip install pywin32
桌面准备（重要！）：桌面图标数量至少需要 5 个（图标越多，蛇能长得越长，解封的“食物”也越多）。
强烈建议：在运行前，关闭桌面的“自动排列图标”功能（在桌面空白处右键 -> 查看 -> 取消勾选“自动排列图标”）。🚀 快速开始三步走启动程序：双击运行脚本或在终端执行 python desktop_snake.py。观察桌面：程序启动后，会瞬间备份你原本的桌面布局。随后，多余的图标会被安全隐藏，桌面上只会留下 3 个图标组成的“初始蛇身”和 1 个作为“食物”的图标。盲操解锁：此时游戏处于静止集结状态。你不需要将鼠标点击切回黑色的终端窗口，直接在任意界面按下 Enter (回车键)，游戏将正式爆发！🕹️ 游戏控制指南本游戏采用了全局异步键盘监听技术，这意味着在游戏过程中，无论你的当前焦点在浏览器、聊天软件还是桌面上，键盘控制均全天候实时生效！按键对应动作备注Enter开始游戏仅在开局摆放好图标后生效，一键打破静止状态。W向上移动蛇头向上挺进（向下移动时按 W 无效，防止自咬）。S向下移动蛇头向下挺进（向上移动时按 S 无效）。A向左移动蛇头向左挺进（向右移动时按 A 无效）。D向右移动蛇头向右挺进（向左移动时按 D 无效）。Q紧急撤退 / 退出随时按下，游戏立刻终止并瞬间完美还原桌面。
